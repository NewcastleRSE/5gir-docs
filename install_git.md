
# Install Git

You need to have Git installed in your system to create the conda environment later. 

Confirm that you have `git` installed in your system, by running the below in your Ubuntu terminal or Windows
Prompt Command.
```bash
git version
```
If the above prints the below, it means that git is not installed, and you need to install it following the instructions
below
```terminaloutput
git: command not found
```

## On Ubuntu

If git is not installed, then you need to install it with the below before creating the conda environment. 
```bash
sudo apt-get install git
```
More info in [here](https://git-scm.com/install/linux)

## On Windows

Download the installer in [here](https://git-scm.com/install/windows). Once download is completed, run the installer.
You can leave all installation options as they are by default and click "Next" in all installation pages and then click
on "Install". Wait for the installation to complete and click on "Finish"

## On Mac
Follow the instructions in [here](https://git-scm.com/install/mac).
